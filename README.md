# Научиться учиться
----------------
Одностраничный сайт на тему обучения. Содержит различные техники и принципы обучения, видео, ссылки на дополнительные ресурсы и книгу про обучение.

# Использованные технологии:
1. Iframe (API Youtube)
2. Flexbox
3. Keyframes
